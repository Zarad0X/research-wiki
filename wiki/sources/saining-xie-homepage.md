## Summary

这条 source 的价值不在某一篇单独论文，而在于它是 Saining Xie 自己给出的研究自画像。官网里的 framing 很清楚：人类和动物的知识主要来自感知经验及与环境的互动，因此走向 human-like intelligence 不能停在 language-only systems，而要走向从连续真实世界感知输入中直接学习的 world models。同时，官网把他过去的重要 work cluster 摆得很明显，从 `Holistically-Nested Edge Detection`、`ResNeXt`、`MoCo`、`MAE`、`ConvNeXt`、`ConvNeXt V2` 到 `DiT`、`Cambrian`，能直接看出他长期在追的并不是窄任务技巧，而是高复用、可扩展、能成为默认基础件的表征与架构。

## Key Takeaways

- 主页最重要的一句自我 framing 是：要超越 language-only systems，转向直接从连续感知输入学习的 world models。
- 这条 source 很适合拿来理解他的 research taste，因为“代表作列表”本身已经透露出稳定偏好：
  - 喜欢能成为 default primitive 的东西，而不是只在单 benchmark 上出彩的技巧
  - 重视 representation，而不只是 task head 或 prompt engineering
  - 重视 architecture simplification 与 scaling，而不是越堆越复杂的系统拼接
- 主页显式列出的代表 work cluster 覆盖了几个连续阶段：
  - 早期视觉结构与边缘表征：`Holistically-Nested Edge Detection`
  - 通用架构设计：`ResNeXt`
  - 自监督表征学习：`MoCo`
  - masked modeling 与可扩展视觉表示：`MAE`、`ConvNeXt V2`
  - 重新审视 ConvNet design space：`A ConvNet for the 2020s`
  - diffusion / generative backbone：`DiT`
  - vision-centric multimodal systems：`Cambrian`
- 主页上的自述、工作列表和奖项信息放在一起看，会让人更容易把他理解成“追求长期高杠杆表征与系统基元的人”，而不是某一篇爆款 paper 的作者。
