# Raw Sources

Store archived source material here.

Guidelines:

- Put converted web pages here once they are worth archiving.
- Papers, interviews, podcast notes, and meeting notes are all fine.
- Prefer one source per file.
- Use dates or short slugs for file names.
- Move material here once it belongs in the long-term research archive.
- Keep this layer stable; avoid frequent rewrites.
- Let these files correspond to the `Sources` sections in `wiki/papers/` or `wiki/sources/`.
- `scripts/ingest_inbox.py` should move structured items here from `inbox/`.
- Use `raw/papers/` for paper-like sources and `raw/sources/` for everything else.

This layer is the source of truth. Agents may read it, but should not casually rewrite its body content.
